Analysis and Simulation of Feedback System

In this subsection, it will be considered the closed loop system of Fig. 4.1.

Example 3.4

Figure 4.8 shows the developed window for analysis and simulation of closed loop system.

Figure 4.8: Main window for analysis and simulation of closed loop system.

Notice that, if the user press load data button then a window like the one in the Fig. 4.9 will be present. In it, the data corresponding to the Fig. 4.1 must be loaded.

Figure 4.9: Window for defining of feedback system transfer function.

On the other hand, the user has a tool for analysis of feedback systems, if the analysis tools button is pressed. Based on the preloaded data in the previous window, this tool provides in a single window with

  1. Poles-Zeros Map of G(s)H(s).
  2. Poles-Zeros Map of the feedback system.
  3. Root locus plot
  4. Bode and Nyquist diagrams
as it is shown in Fig. 4.10.

Figure 4.10: Tool Window for analysis of feedback system.

Furthermore, the user has a simulation parameter window where allows setting step amplitude and initial time of set-point and disturbance changes and, the simulation time.

Figure 4.11: Tool window for defining simulation parameters.

Finally, in Fig. 4.12 the simulation of this example is shown.

Figure 4.12: Window with a simulation example.
